
# na-urmborder – LaTeX package for Urmand-style artistic borders

## Description
The `na-urmborder` package provides a collection of professional and artistic page borders based on the **Urmand** font style. Developed using **TikZ**, this package is specifically designed for educational materials, certificates, and mathematical magazines (such as Al-Khalil). 

It is fully compatible with **XeLaTeX** and supports Arabic typesetting through **polyglossia**, ensuring high-quality rendering for both right-to-left and left-to-right documents.

## Features
- Artistic frames inspired by Urmand decorative patterns.
- Easy integration with standard LaTeX document classes.
- Optimized for educational layouts and exam papers.
- Supports Arabic language and RTL (Right-to-Left) direction.

## Files Included
- `na-urmborder.dtx`: The documented source file.
- `na-urmborder.ins`: The installation file.
- `na-urmborder.pdf`: The package documentation.
- `test-xelatex.tex`: A sample file demonstrating the borders.
